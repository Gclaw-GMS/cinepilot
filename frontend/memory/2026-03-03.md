- This is a data/model issue - characters from character-costume aren't linked to script scenes

**Solution Implemented:**
1. Enhanced DOOD API to detect unlinked characters state

   - Shows amber warning when characters aren't linked
   - Provides "Extract & Link Characters" button
   - Provides "Go to Script Breakdown" link
   - Includes "Dismiss" option
